Random Thoughts

Motivation:

  • manual maintenance of Canvas Rubric with really assignment especially very tedious step-by-step tutorial is too error prone
    • Need to write my own code allowing HTML5 assignment/tutorial description to automatically generate Rubrics with forward/backward links and decent tools for semi-automating calculations especially for tutorials which involve many specific step-by-step instructions
    • Need to be agnostic about using a particular LMS because we change LMS every few years and re-writing and porting assignment and rubrics is waste of time. better to use standard digital media tools.   Past use of MS Word didn't allow for good auto calculations unless I wanted to waste my time using priorietary MS Word macro programing or their annoying Visual Basic
  • Writing assignment directly in HTML5 allows use of WebGL and SVG for all illustrations, no more proprietary MS PowerPoint drawings, etc.

Risks

  • This pile of effort must turn out to be very reusable across furture projects and years or this is going to have been a big waste of time
